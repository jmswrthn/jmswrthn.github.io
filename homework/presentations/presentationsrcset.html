SRCSET and SIZES attributes

What they do

SRCSET and SIZES creates a Responsive Image

Responsive images don't change just because of a change in envirement

Use a combination of HTML and CSS to display different images in the different envirements

Web sites that use it

https://raventools.com

how it is coded

SRCSET

First we need the same images in different resolutions for ex.) small med and large

SRCSET can also run alone by using display density descriptor

We include the original src at the beginning for backwards compatibility in case the browser does not support SRCSET

listing the different images followed by the screen resolution followed by a ',' with each extra img

SIZES

after selecting images of different resolutions we then need to specify the size of the images, this is similar to the media queries in practice.

For example sizes is coded like so
sizes="[media query] [length], [media query] [length] so forth"

What role they play

The common use of using SRCSET and SIZES is to allow web pages to load as quick as possible

No one wants to load an image to the mobile browser that is so large that it kills your data

Or do they want to load a small image that looks like a thumbnail on their large screen
